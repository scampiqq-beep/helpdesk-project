# Automation Engine — Step 1 (Foundation)

Этот шаг добавляет безопасный фундамент под движок правил автоматизации.

Что входит:
- доменные типы правил и действий
- сервис применения правил к тикету
- in-memory evaluator без обязательной миграции БД
- журнал срабатываний в памяти ответа

Важно:
- этот шаг не включает UI админки и не включает автоматический запуск на все события
- шаг безопасен для текущего runtime, потому что не меняет существующие маршруты и шаблоны
- следующие шаги:
  1. wiring на create/update ticket
  2. admin UI
  3. фоновые правила и эскалации
