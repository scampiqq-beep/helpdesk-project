# Automation Engine — Step 3 (Admin UI)

Этот шаг подготавливает UI и admin-layer для правил автоматизации.

Что входит:
- service для страницы правил
- route-level foundation для admin automation
- шаблоны списка и редактора правил
- безопасный mock-storage через JSON-файл в instance/

Что НЕ входит:
- полноценные миграции БД
- фоновые правила
- продвинутый visual builder
